3 commands:

!memehelp - shows other 2 commands

!templates - shows all possible meme templates

!cmeme - Creates a meme!

Syntax for !cmeme: !cmeme <id> <arg1>, <arg2>, <arg3>, <arg4>, ...

In the list of templates, each template will have:

  • Meme template name (blue text)
  • Template id (first number)
  • Number of captions in the meme (second number)

Seperate all captions by commas.
